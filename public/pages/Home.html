What is s16?
s16 is a system written in JavaScript that mimics an old computer system. It doesn't try to emulate any existing architecture or operating system, it simulates a fictional 16-bit system which is designed specifically to have a simple and retro feel - all will become clearer when s16is fully functional and I can move on to phase 2

The goal is to build a programmable platform that will provide the foundations for a future project - for now, a demo of s16 will exist here on this site, anyone interested may interact with s16, browse the code, maybe even offer suggestions or improvements.

Hover over the nagivation bar to the left of the page and it should automatically expand, there you will find a link to the Terminal, this is where you can interact with the s16 system.

You will also find a link to some Documentation that will explain how the system works on a more technical level.
